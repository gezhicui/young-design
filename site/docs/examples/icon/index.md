# Icon 图标

## 基础用法

<y-icon name="zuo" />

```vue
<template>
  <y-icon name="zuo" />
</template>
```

## 修改颜色

<y-icon name="you" color="red" />

```vue
<template>
  <y-icon name="you" color="red" />
</template>
```

## 添加角标

<y-icon name="you" dot />

```vue
<template>
  <y-icon name="you" dot />
</template>
```

## 数字角标

<y-icon name="yixuanze" dot badge="1" />

```vue
<template>
  <y-icon name="yixuanze" dot badge="1" />
</template>
```
