# young-design

基于 vue3 setup+monorepo+vite 实现的组件库

文档地址 ： [github page 部署地址](https://gezhicui.github.io/)

文档地址 ： [gitee page 部署地址(国内较快)](http://xiang0515.gitee.io/young-design-docs/)

# 启动

## 安装 pnpm

> npm i pnpm -g

## 安装所有依赖

> pnpm install

## 打包组件

> pnpm build

## 本地组件测试

> pnpm start

## 启动文档

> pnpm start:docs
